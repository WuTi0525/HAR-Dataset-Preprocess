## Human Activity Recogniton (HAR) 公开数据集预处理与网络搭建

### 包含数据集
* phyphox软件自采集excel数据
* Daily-and-Sports-Activities-dataset
* PAMAP2 dataset
* UCI-HAR dataset
* USC-HAD dataset
* UniMiB-SHAR dataset
* WISDM dataset
* OPPORTUNITY dataset (unfinished)

### 代码运行样例

$ cd UniMiB-SHAR
$ python dataproc.py
